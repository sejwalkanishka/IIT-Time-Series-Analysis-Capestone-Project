# 📈 Time Series Forecasting & Portfolio Optimization Dashboard

This project focuses on forecasting NSE-listed stock prices using **ARIMA** and **LSTM** models and constructing a diversified investment portfolio using volatility-aware allocation strategies. The project was developed as part of the **Time Series Analysis Capstone Project 2026** in collaboration with StockGro.

---

# 🚀 Project Overview

The project combines:

- Time Series Forecasting
- Deep Learning
- Portfolio Optimization
- Risk Analysis
- Interactive Dashboarding

to simulate real-world stock market analysis and portfolio management.

---

# 📌 Key Features

✅ Stock price forecasting using ARIMA and LSTM  
✅ Model evaluation using RMSE, MAE, MAPE & Directional Accuracy  
✅ Volatility and risk analysis  
✅ Portfolio optimization using forecast-guided allocation  
✅ Sharpe Ratio analysis  
✅ Predicted vs Actual price comparison  
✅ Interactive Streamlit dashboard  
✅ Execution strategy visualization  

---

# 🧠 Models Implemented

| Model | Purpose |

| LSTM | Deep Learning-based Sequential Forecasting |
